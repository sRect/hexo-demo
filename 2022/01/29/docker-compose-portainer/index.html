报错部分：

一番折腾（又是删除 node_modules 重新 install 安装，又是升级依赖到最新版本，又是检查文件夹路径有无中文…）过后，找到了解决方法：

4.5 注意事项

  • 数据库连接配置时区timezone，设置为东 8 区

7.1 Portainer 的好处

这就如同 git，使用 git GUI 工具和 git bash 命令行其他一样，方便容器管理操作。如果你喜欢命令行，就直接跳过这一节，直接部署容器吧。
